# NLP-Exercise-2
Character-based language models: generating Catalan names across two time periods. Our two models predict Catalan names based on a 1997 dataset and on a 2024 dataset each. The code is based on Andrej Karpathy's makemore model. The code for both models is in the .ipynb files called model1997 and model2024. A simple accuracy is measured in the metrics.py file.
# Dependencies
Project made in Google Colab, so the Python version used is 3.11. Libraries used:
- torch
- matplotlib.pyplot
